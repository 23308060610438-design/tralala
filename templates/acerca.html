Acerca del Proyecto

Este navegador web fue desarrollado con Flask y Bootstrap como parte de un proyecto educativo y creativo. Su propósito es mostrar contenido visual y multimedia sobre temas fascinantes del mundo.

  • Inicio: Página principal con título personalizado y bienvenida visual.
  • Animales: Galería de animales exóticos con imágenes reales y descripciones breves.
  • Vehículos: Colección de vehículos antiguos con estilo clásico y datos históricos.
  • Maravillas: Las 7 maravillas del mundo moderno presentadas en tarjetas con video local.
  • Acerca de: Esta misma sección que resume el contenido del sitio.

Cada sección está diseñada para ser visualmente atractiva, responsiva y fácil de navegar. El proyecto combina HTML, Bootstrap y Flask para lograr una experiencia web modular y funcional.
